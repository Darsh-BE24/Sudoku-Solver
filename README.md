# Sudoku Solver in Python

A command-line Sudoku solver built in Python using recursive backtracking. This project solves any valid 9x9 Sudoku puzzle and prints both the unsolved and solved boards in a clear and readable format.

---

## Features

- Solves standard 9x9 Sudoku puzzles
- Uses recursive backtracking algorithm
- Validates rows, columns, and 3x3 subgrids
- Clearly displays the puzzle before and after solving
- Pure Python — no external libraries required
